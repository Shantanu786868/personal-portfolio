DESCRIPTION : This project is a **modern and responsive personal portfolio website** developed using **React, TypeScript, Framer Motion, and custom CSS**. It is designed for a full-stack developer to professionally showcase their profile, technical skills, featured projects, and contact information in a visually appealing and user-friendly manner. 

The website follows a **single-page portfolio structure** with smooth scrolling navigation between different sections such as:

* Home/Hero section
* About section
* Skills section
* Featured Projects section
* Contact section

The interface uses a clean and minimal design with elegant typography, soft terracotta and cream color combinations, hover effects, and smooth animations to create a premium modern look. 

The frontend is built with:

* **React + TypeScript** for component-based development
* **Framer Motion** for animations and transitions
* **React Icons & Lucide Icons** for modern icon integration
* **Plain CSS** for custom responsive styling

The project includes several interactive features such as:

* Fixed responsive navigation bar
* Mobile hamburger menu
* Smooth scrolling between sections
* Animated content reveal effects
* Responsive project cards
* Skill showcase cards
* Social media links
* Functional contact form

The portfolio also contains a backend-integrated contact system. When a user submits the contact form, the frontend sends the data to an Express API route, which forwards the request to a PHP backend for validation and processing. Submitted messages are then stored in a log file.  

The Featured Projects section highlights sample projects like:

* TaskFlow
* WeatherNow
* DevBlog
* ShopCart

Each project card displays:

* Project title
* Short description
* Technology stack
* GitHub and live demo links

The design is fully responsive, ensuring proper display across desktops, tablets, and mobile devices. Overall, this project demonstrates strong skills in:

* Frontend development
* Responsive web design
* UI/UX implementation
* Animation handling
* API communication
* Form validation
* Modern portfolio design principles
